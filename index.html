Ejemplo A : Posicion Static

En el ejemplo A , puedes ver tres elementos apilados como una simple torre. Fascinante, ¿no es así? Esta es la construcción de bloques 101. ¡Felicitaciones! Puede usar el staticvalor para diseños simples de una sola columna donde cada elemento debe estar encima del siguiente. Si quieres empezar a reorientar los elementos alrededor de usar propiedades de desplazamiento tales como top, right, bottom, y left, estás de suerte. Estas propiedades no están disponibles para un staticelemento. De hecho, un staticelemento ni siquiera puede crear un nuevo sistema de coordenadas para elementos secundarios. Espere. ¿Qué? Me perdiste en el sistema de coordenadas . Roger eso, Roger. Vamos a explicar usando el relativevalor. Los elementos posicionados relativamente se comportan como los elementos posicionados estáticamente; Juegan bien con los demás, se amontonan bien y no causan alboroto. Es difícil de creer, ¿verdad? Ejemplo C : En el relative se puede modificar la posicion

El ejemplo C muestra este relativeposicionamiento en acción. Nuestros tres bloques están apilados muy bien, pero esta vez el bloque azul ( #box_2) se expulsa 200 píxeles desde la izquierda. Aquí es donde comenzamos a doblar la ley de la gravedad a nuestra voluntad. El bloque azul todavía está en el flujo del documento, los elementos se están apilando uno encima del otro, pero noten el bloque verde ( #box_3) en la parte inferior. Está sentado debajo del bloque azul, aunque el bloque azul no está directamente encima de él. Cuando usa la propiedad de desplazamiento para desplazar un elemento relativamente posicionado, no afecta los elementos que siguen. El cuadro verde todavía se posiciona como si el cuadro azul estuviera en su posición no compensada. Intenta eso con tu torre de bloques alfabeto.

Ejemplo D : mover la caja azul

El ejemplo D muestra nuestro nuevo marcado. Debido al nuevo sistema de coordenadas, el bloque azul mide su desplazamiento a 200 píxeles de la izquierda del bloque rojo ( #box_1) en lugar del documento. Esta práctica es más común con los elementos establecidos position: absolute, como usted desea que haya construido torres cuando era más joven.

Ejemplo E : Position Absolute

El ejemplo E muestra cuatro cuadros, cada uno en una esquina de la ventana del navegador. Desde que establecemos el positionvalor de cada caja absolute, básicamente hemos velcro un cuadro a cada esquina de nuestra ventana del navegador. Al cambiar el tamaño del navegador, esos cuadros permanecerán en sus esquinas respectivas. Si reduce la ventana del navegador para que los recuadros se superpongan, notará que no hay interacción en absoluto; eso se debe a que están fuera del flujo normal del documento. Al igual que los relativeelementos, los absoluteelementos crean un nuevo sistema de coordenadas para elementos secundarios.
